
# Nikon Z FileSort

This is a batch file that separates photos taken with Nikon Z5 into RAW and JPEG.

Camera settings are default.


## How to use

1. Initial State
```
├ filesort.bat
├ photo1.JPG
├ photo1.NEF
├ photo2.JPG
├ photo2.NEF
└ ...
```

2. Click filesort.bat

3. After State
```
├ filesort.bat
├ JPEG
│   ├ photo1.JPG
│   └ ...
└ NEF
    ├ photo1.NEF
    └ ...
```
